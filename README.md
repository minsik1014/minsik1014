## 🚀 Key Projects

### 🎮 PlayProof (게임 팀워크 관리 플랫폼)
- **Period**: 2025.09 - 2025.11 (UMC 9기)
- **Role**: **프론트엔드 팀장** (아키텍처 설계 및 핵심 기능 구현)
- **Key Achievements**:
    - **UMC 데모데이 우수상 수상**
    - **Feature-based 아키텍처**를 도입하여 대규모 기능(채팅, 스케줄러, 상점 등)의 유지보수성을 극대화했습니다.
    - TanStack Query와 Zustand를 조합해 서버/클라이언트 상태를 명확히 분리하고 동적인 UI 로직을 설계했습니다.
- [GitHub Repo](https://github.com/Playproof-Umc/Playproof-Frontend)

### 🛡️ GLife (산업안전교육 플랫폼)
- **Period**: 2025.08 - 2025.10
- **Role**: 대시보드 UI 및 JWT 인증 시스템 구현
- **Key Features**:
    - **JWT Refresh 플로우**와 보호 라우트를 설계하여 보안성 있는 인증 시스템을 구축했습니다.
    - 엑셀 업로드 및 헤더 검증 기능을 구현하여 관리자의 데이터 관리 편의성을 높였습니다.
    - Mock API를 활용해 백엔드 없이도 전체 서비스 플로우를 검증했습니다.
- [GitHub Repo](https://github.com/minsik1014/GLife_EXPO_React)

### ☁️ ChildOfWeather (날씨 기반 활동 관리)
- **Role**: 기획 및 풀스택 개발
- **Tech**: Java, Servlet/JSP, JDBC, MySQL
- **Description**: 기상청 API와 네이버 지도를 연동한 실시간 위치 기반 서비스입니다. MVC 패턴을 직접 구현하며 서버 렌더링과 DB 흐름을 깊게 이해하는 계기가 되었습니다.
- [GitHub Repo](https://github.com/JeongGyul/child-of-weather)

### 📊 Pokemon Data Dashboard
- **Tech**: Python, Streamlit, Pandas
- **Description**: 포켓몬 데이터셋을 분석하여 타입별 능력치 분포와 관계를 시각화한 대시보드입니다. 데이터 전처리부터 인터랙티브 UI까지 단독으로 구현했습니다.
- [GitHub Repo](https://github.com/minsik1014/bigData_Pokemon)
